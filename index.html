L0gVi3w

L0gVi3w is a tail implementation in PHP, specialized in PHP log files.

Download

You have two options (for now) :
  • Check out the project from GitHub: 
    git://github.com/elkuku/L0gVi3w.git
OR
  • Click the "Download" button on the projects git Page.

Installation

Installation is easy - just copy or symlink the two folders to their proper locations: 
  • component/site/com_l0gvi3w
    ⇒ 
    J/components/com_l0gview
  • and
  • component/admin/com_l0gvi3w
    ⇒ 
    J/administrator/components/com_l0gview
You may now install the extension using Joomla!s discover_install.

Usage

Please note that to actually "see something" the following options must be set correctly in your php.ini file:
  1. log_errors must be set to true.
  2. error_log must point to a valid (and accessible) file.
